##  Project Description

This project is part of the Open Source Software capstone. It includes analysis of MySQL as an open-source software along with implementation of five shell scripts demonstrating Linux concepts and automation.

---

##  Shell Scripts Description

### 1. script1.sh — System Identity Report

Displays system information such as kernel version, logged-in user, uptime, and date using variables and command substitution.

### 2. script2.sh — FOSS Package Inspector

Checks whether MySQL (or another package) is installed. Displays version, license, and summary using system commands and a case statement.

### 3. script3.sh — Disk and Permission Auditor

Analyzes important directories like `/etc`, `/home`, etc., and shows their size, permissions, owner, and group using loops and Linux commands.

### 4. script4.sh — Log File Analyzer

Reads a log file and counts occurrences of a keyword like "error". Uses loops, conditionals, and command-line arguments.

### 5. script5.sh — Open Source Manifesto Generator

Takes user input and generates a personalized open-source statement, saving it into a text file.

## ▶ How to Run

### Step 1: Give execution permission

```bash
chmod +x *.sh
```

### Step 2: Run scripts

```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
```
